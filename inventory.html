Setup Guide

Google Sheets (5 mins):
1. Create a new Google Sheet and share it (Editor access) with everyone who needs it.
2. In the Sheet, go to Extensions → Apps Script.
3. Delete any existing code and paste in the Apps Script from the separate file provided.
4. Click Deploy → New deployment → Web app. Set "Who has access" to Anyone. Click Deploy.
5. Copy the Web App URL and paste it into the Apps Script URL field above.

Telegram:
Message @BotFather → /newbot → get your token. Add the bot to your channel as admin. Get Chat ID from the getUpdates URL.
